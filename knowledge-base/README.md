# Holistic Wellness Knowledge Base - Quick Reference Guide

## For AI Processing

When answering wellness questions, follow this priority:

1. **Term Lookup**: Check [GLOSSARY.md](GLOSSARY.md) for definitions
2. **Query Routing**: Use [NAVIGATION.md](NAVIGATION.md) to find relevant files
3. **Quick Answers**: Check [FAQ.md](FAQ.md) for common questions
4. **Deep Dive**: Navigate to topic folders for detailed content

### File Organization

- `/concepts/` - Core wellness principles, philosophy, foundational ideas
- `/protocols/` - Step-by-step methods, routines, procedures
- `/case-studies/` - Real examples, patient stories, before/after
- `/templates/` - Fill-in-the-blank guides, worksheets
- `/checklists/` - Action lists, daily/weekly tracking
- `/reference/` - Diagrams, recommended reading, external resources
- `/data/` - Logs, tracking data, research notes
